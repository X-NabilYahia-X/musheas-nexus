# 🍄 MUSHEAS Nexus - Biotech Dashboard

**MUSHEAS Nexus** هو النظام الإداري المتكامل لشركة **MUSHEAS**، أول شركة جزائرية متخصصة في إنتاج مستخلصات الفطر والمواد الأولية التجميلية الحيوية.

## 🚀 نظرة عامة
تم بناء هذه المنصة باستخدام **Next.js 15** لتوفير واجهة سريعة واحترافية لإدارة عمليات المختبر، طلبات العينات، وتتبع الشحنات التقنية لعملاء الـ B2B.

## ✨ الميزات الرئيسية
* **إدارة عينات B2B:** نظام مخصص لمتابعة طلبات العينات المخبرية للمؤسسات.
* **تتبع الشحنات:** تكامل مع مزودي الشحن لتوليد أرقام التتبع تلقائياً.
* **سجل العمليات (Audit Logs):** مراقبة دقيقة لكل حركة تتم داخل النظام لضمان الجودة.
* **واجهة مستخدم متطورة:** تصميم عصري مستوحى من الطبيعة والبيوتكنولوجيا باستخدام Tailwind CSS.

## 🛠 التكنولوجيات المستخدمة
* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Styling:** Tailwind CSS & Shadcn/UI
* **Icons:** Lucide React
* **Deployment:** Vercel
* **State Management:** React Hooks & Context API
